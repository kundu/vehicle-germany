### Follow below steps to run the project

- git clone
- create database and database user, password
- rename env.example to .env
- update the database credential values
> *DB_DATABASE=vehicle_db
DB_USERNAME=root
DB_PASSWORD=password*
- run command: 
> **composer install**
> **php artisan migrate**
